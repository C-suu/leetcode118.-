```python
把杨辉三角的每一排左对齐：
[1]
[1,1]
[1,2,1]
[1,3,3,1]
[1,4,6,4,1]
设要计算的二维数组是 c，计算方法如下：
每一排的第一个数和最后一个数都是 1，即 c[i][0]=c[i][i]=1。
其余数字，等于左上方的数，加上正上方的数，即 c[i][j]=c[i−1][j−1]+c[i−1][j]。例如 4=1+3, 6=3+3 等。

class Solution:
    def generate(self, numRows: int) -> List[List[int]]:
        c = [[1] * (i + 1) for i in range(numRows)]  # 初始化一个全部由 1 组成的二维列表，每行长度递增
        for i in range(2, numRows):                  # 从第三行（下标2）开始逐行向下遍历
            for j in range(1, i):                    # 遍历当前行的内部元素（避开下标0和下标i的边缘1）
                # 核心公式：当前元素 = 上一行左侧元素 + 上一行同列元素
                c[i][j] = c[i - 1][j - 1] + c[i - 1][j]  # 左上方的数 + 正上方的数
        return c                                     # 计算完毕，返回完整的杨辉三角

```


以下是针对《118. 杨辉三角》这道题的详细拆解：

### （1）用白话文解释题目意思及生僻概念

**题目白话文解释：**
需要按照特定规律搭建一个数字金字塔（共有 `numRows` 层）。
搭建规则如下：
1. 金字塔的最顶端（第一层）只有一个数字 `1`。
2. 每一层的最左侧边缘和最右侧边缘的数字永远是 `1`。
3. 从第三层开始，夹在边缘中间的那些数字，其数值等于**它左上方数字与右上方数字相加的和**。
目标是把这个金字塔按层打包成一个列表的列表（二维数组）并返回。

**生僻概念解释：**
* **杨辉三角**：就是上述规则构成的数字三角形。在西方称为帕斯卡三角形。它的特点是除了边缘，内部的每一个节点都是上方两个节点的和。
* **二维数组（列表的列表）**：在代码中，金字塔并不是真正的等腰三角形排列，而是靠左对齐排列的梯队。例如第三行 `[1, 2, 1]`，它在数组中的实际位置是：
  ```text
  第0行: [1]
  第1行: [1, 1]
  第2行: [1, 2, 1]
  ```
  因此，视觉上的“左上方”对应上一行的前一个位置（下标 `j-1`），视觉上的“右上方”对应上一行的正上方位置（同一下标 `j`）。

### （2）解题思路的来源（基于代码反推）

这段代码采用了**“先打好地基，再填补空缺”**的动态规划思路。

1. **预分配内存与初始化边界**：既然已知每一层的第一个数和最后一个数都是 `1`，与其每次计算时去判断是不是到了边界，不如一开始就把整个金字塔全部填满 `1`。第一层 1 个 `1`，第二层 2 个 `1`，依此类推。这样，所有的边界条件天然就满足了。
2. **定位需要计算的内部区域**：全填成 `1` 之后，前两层（第 0 行和第 1 行）就已经完全正确了，不需要修改。真正需要修改的是从第 3 层（下标为 2）开始，并且每层只修改“掐头去尾”的内部元素。
3. **状态转移（套用公式）**：利用两层循环，外层控制“在哪一层”，内层控制“修改哪一个内部元素”。根据对齐后的坐标映射规律，当前位置的值等于**上一行同一列的值**加上**上一行前一列的值**。逐个覆盖掉原本占位的 `1`，即可得到最终结果。

### （3）每一行代码的注释

        c = [[1] * (i + 1) for i in range(numRows)]  # 初始化一个全部由 1 组成的二维列表，每行长度递增

### （4）每一行代码的详细中文解释

* `class Solution:`
    * 定义封装解题算法的类。
* `def generate(self, numRows: int) -> List[List[int]]:`
    * 定义函数 `generate`，接收一个名为 `numRows` 的整数（代表生成的行数），并指明最终返回一个包含多个整数列表的列表（二维数组）。
* `c = [[1] * (i + 1) for i in range(numRows)]`
    * 利用列表推导式，一步到位地构建出整个三角形的基础骨架。`range(numRows)` 会生成从 `0` 到 `numRows - 1` 的行号 `i`。对于每一行，生成一个长度为 `i + 1` 且全部填充为 `1` 的列表。生成的二维列表赋值给变量 `c`。
* `for i in range(2, numRows):`
    * 开启外层循环，变量 `i` 代表当前正在处理的行号。由于第 0 行和第 1 行全为 1，不需要任何计算，所以直接从下标 `2`（即第三行）开始遍历，直到最后一行。
* `for j in range(1, i):`
    * 开启内层循环，变量 `j` 代表当前处理到这一行的哪个位置（列号）。对于下标为 `i` 的行，它的长度是 `i + 1`，合法的下标是 `0` 到 `i`。为了避开首尾的 `1`，循环范围设定为从 `1` 到 `i - 1`（`range(1, i)` 在 Python 中不包含 `i`）。
* `c[i][j] = c[i - 1][j - 1] + c[i - 1][j]`
    * 执行加法并更新数值。将二维数组 `c` 中，上一行（`i - 1`）前一列（`j - 1`）的数值，与上一行（`i - 1`）同一列（`j`）的数值相加，得到的和覆盖掉当前位置 `c[i][j]` 原本占位的 `1`。
* `return c`
    * 当所有的嵌套循环结束时，二维数组内部该替换的数值已经全部替换完成，完整的杨辉三角构建成功，将其返回。
